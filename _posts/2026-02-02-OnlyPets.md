## 1. 개요

### 목적  

반려동물을 키우는 사람들을 위한 SNS·커뮤니티·서비스 통합 모바일 앱입니다.  
사진/영상 피드, 숏폼, 강의, 펫시터 예약·결제까지 한 앱에서 제공합니다.  
Flutter 기반으로 Android / iOS를 지원합니다.

## 4. 기술

### 4.1 스택

* **Framework**: Flutter (Dart SDK ^3.8.1)
* **상태관리 / 라우팅 / DI**: GetX (`get`)
* **네트워크**: `dio`
* **인증/푸시**: Firebase (Auth, Cloud Firestore, Messaging), Google Sign-In
* **로컬 저장소**: `shared_preferences`, `get_storage`
* **미디어 처리**: `video_player`, `chewie`, `video_editor`, `ffmpeg_kit_flutter_new`, `video_compress`, `image_picker`, `image`, `cached_network_image`
* **기타**: `geolocator`(위치), `table_calendar`(예약 캘린더), `flutter_local_notifications`, `flutter_rating_bar`, `permission_handler`

### 4.3 주요 기능

#### 1. 인증 / 계정

* 이메일·비밀번호 로그인 / 회원가입
* 자동 로그인 및 Refresh Token 기반 액세스 토큰 갱신
* Firebase 인증 및 FCM 토큰 발급
* 비밀번호 변경, 프로필 편집

#### 2. 피드 (Feed)

* 사진/동영상 게시물 업로드 및 상세 보기
* 좋아요, 댓글/대댓글, 태그
* Pull-to-refresh를 이용한 무한 스크롤 피드
* 게시물 공유 / 신고

#### 3. 숏폼 (Shorts)

* 세로 스크롤 형식의 짧은 동영상 플레이어
* 영상 업로드 시 컷 편집 / 압축 (`video_editor`, `ffmpeg_kit_flutter_new`, `video_compress`)
* 자동 재생 / 좋아요 / 댓*

#### 4. 업로드

* 업로드 타입 선택 (피드 게시물 / 숏폼)
* 다중 이미지 선택 및 미리보기, 영상 트리밍
* 업로드 후 게시물 수정(Update)

#### 5. 탐색 (Explore)

* 추천/인기 게시물 카드형 탐색
* 사용자 검색 및 다른 사용자 프로필 조회
* 팔로우 / 팔로잉 목록 관리

#### 6. 마이페이지

* 내 프로필, 게시물·숏폼 그리드 조회
* 설정 (알림, 비밀번호 변경 등)
* 내가 결제한 강의 목록 / 내 예약 목록

#### 7. 서비스 — 강의 (Lecture)

* 카테고리(테마)별 강의 탐색 및 상세 조회
* 강의 검색
* 강의 결제 및 결제 결과 화면
* 강사 등록 / 강사 인증
* 강사 대시보드: 강의 업로드, 강의 상세 관리, 테마 생성
* 수강 신청 후 진도(progress) 관리

#### 8. 서비스 — 펫시터 (Pet Sitter)

* 펫시터 등록 / 인증
* 펫시터 대시보드: 운영 시간 관리, 예약 관리
* 사용자: 펫시터 검색·상세 보기, 예약, 리뷰 작성
* 펫시터 ↔ 보호자 간 1:1 채팅 (Cloud Firestore 기반)
* 결제 화면

#### 9. 알림 (FCM)

* Firebase Cloud Messaging 포그라운드/백그라운드 메시지 처리
* 로컬 알림 표시 (`flutter_local_notifications`)

#### 10. 권한 / 위치

* 카메라, 사진/영상, 알림, 위치 등 권한 요청 (`permission_handler`)
* 펫시터 검색 등에 필요한 위치 정보 (`geolocator`)
